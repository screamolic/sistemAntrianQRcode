## Objective

Set up TypeScript path aliases and create organized project structure.

---

## Tasks

### 1. Configure TypeScript Path Aliases
- [ ] Update `tsconfig.json` with paths:
  ```json
  {
    "compilerOptions": {
      "baseUrl": ".",
      "paths": {
        "@/*": ["./src/*"],
        "@/components/*": ["./src/components/*"],
        "@/lib/*": ["./src/lib/*"],
        "@/types/*": ["./src/types/*"],
        "@/hooks/*": ["./src/hooks/*"],
        "@/app/*": ["./src/app/*"]
      }
    }
  }
  ```
- [ ] Verify Next.js `next.config.js` supports aliases (automatic in Next.js 15)

### 2. Create Directory Structure

### 3. Create Utility Files
- [ ] Create `src/lib/utils.ts` with cn() helper:
  ```typescript
  import { clsx, type ClassValue } from "clsx"
  import { twMerge } from "tailwind-merge"
  
  export function cn(...inputs: ClassValue[]) {
    return twMerge(clsx(inputs))
  }
  ```
- [ ] Create `src/types/index.ts` with base types

### 4. Create Base Layout
